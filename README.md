## Getting Started

Task 1-8 are simple HTML + CSS.
1. Open `index.html` in your browser to see the result.
4. Study the code and comments in `index.html` and `style.css`.

Task 9-15 are complete Vite + Tailwind CSS projects. To run any task:
```bash
cd task-XX-name
npm install
npm run dev
```
Then open `http://localhost:5173` in your browser.

## Learning Path

1. Start with **Task 09** to learn Tailwind basics
2. Progress through **Tasks 10-12** to master components
3. Build complex layouts in **Tasks 13-14**
4. Integrate everything in **Task 15**

## Key Concepts Covered

- **Utility-First CSS**: Using Tailwind classes instead of writing CSS
- **Responsive Design**: Mobile-first approach with breakpoints
- **Dark Mode**: Supporting light and dark themes
- **Component Patterns**: Building reusable UI components
- **Animations**: Smooth transitions and interactions
- **Best Practices**: Professional, production-ready code

## Why Tailwind?

Tailwind CSS is the industry standard for modern web development. It enables rapid development, consistency, and maintainability. Learning Tailwind is essential for any frontend developer.
