# Task 07: Testimonial Carousel (CSS only)

## Introduction
Social proof is vital for SaaS. A carousel allows you to show multiple testimonials in a limited space.

## Learning Points
- **CSS Animation**: Using `@keyframes` to create motion without JavaScript.
- **Overflow Hidden**: How to hide elements that are outside their parent container.
- **Pseudo-elements**: Using `::before` to add decorative quotation marks.
- **Infinite Loop Logic**: How to structure slides so they appear to loop forever.

## Why this matters for SaaS
Testimonials build trust. An animated carousel catches the eye and keeps the user engaged with your success stories.

## How to run
1. Open `index.html`.
2. Wait a few seconds and watch the testimonial automatically slide to the next one.
3. Study the `@keyframes` in `style.css` to see how the timing works.
