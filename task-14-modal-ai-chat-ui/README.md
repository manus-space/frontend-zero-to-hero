# Task 14: Modal + AI Chat Bubble UI

## Introduction
This task teaches you how to build interactive modals with chat interfaces. You'll learn about modal animations, chat bubble layouts, and scroll behavior. This is essential for building modern SaaS features like AI assistants.

## Learning Points
- **Modal Animation**: Using Tailwind's `animate-in`, `fade-in`, and `zoom-in` for smooth transitions.
- **Chat Layout**: Building user vs AI message bubbles with different alignments.
- **Scroll Behavior**: Managing overflow and scroll in modal content.
- **Transitions**: Creating smooth open/close animations.
- **Backdrop Blur**: Using `backdrop-blur-sm` for modern modal effects.

## Why this matters for SaaS
AI chat interfaces are becoming standard in SaaS. Learning to build chat UIs is a valuable skill. Modals are also essential for alerts, forms, and user interactions.

## How to run
```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser and click "Open AI Chat" to see the modal.

## Key Features
- Smooth modal open/close animations
- Chat bubble UI with user/AI distinction
- Typing indicator animation
- Scrollable chat history
- Input area with send button
- Dark mode support
- Backdrop blur effect
