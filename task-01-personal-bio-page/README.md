# Task 01: Personal Bio Page

## Introduction
This is the very first step in your frontend journey! As a Java/Spring Boot developer, you're used to logic and data. Here, we focus on **structure** and **presentation**.

## Learning Points
- **Semantic HTML5**: Using tags like `<header>`, `<main>`, `<section>`, and `<footer>` instead of just `<div>`. This helps browsers and search engines understand your content.
- **Section Structure**: Organizing content into logical blocks.
- **Image & Text Alignment**: Learning how to place images alongside text using Flexbox.
- **Basic Responsiveness**: Using Media Queries to make the page look good on mobile devices.

## Why this matters for SaaS
Every SaaS product needs an "About Us" page or a "User Profile" section. Understanding how to structure basic information is the foundation of building any dashboard or landing page.

## How to run
1. Simply open the `index.html` file in any web browser (Chrome, Firefox, Edge).
2. Right-click and select "Inspect" to see how the CSS affects the HTML elements.
