# Task 26: ES6 Modules + Calculator

## Learning Points

This task teaches you essential concepts for building modern web applications.

## How to Run

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

This task covers important JavaScript/TypeScript patterns used in production SaaS applications.

Understanding these concepts will make you a better frontend developer and prepare you for learning React and other frameworks.
