# Task 11: SaaS Pricing Section with Toggle

## Introduction
This task teaches you how to build a professional pricing page with a monthly/yearly toggle. You'll learn about state-like UI patterns, popular plan highlighting, and interactive price updates.

## Learning Points
- **State-like UI**: Creating toggle switches that change pricing display.
- **Popular Plan Highlight**: Using `md:scale-105`, borders, and gradients to emphasize the recommended plan.
- **Responsive Pricing Cards**: Building three-column layouts that stack on mobile.
- **Interactive JavaScript**: Updating prices dynamically when the toggle is clicked.
- **Visual Hierarchy**: Using colors, badges, and sizing to guide user attention.

## Why this matters for SaaS
Pricing pages are critical for revenue. A well-designed pricing section with monthly/yearly toggle can increase conversions significantly.

## How to run
```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser and try toggling between monthly and yearly pricing.

## Key Features
- Interactive billing period toggle
- Three pricing tiers with different features
- Popular plan highlighted with scale and border
- Price updates on toggle
- Dark mode support
- Fully responsive design
