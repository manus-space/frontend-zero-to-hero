# Task 08: Dashboard Layout Skeleton

## Introduction
The "Dashboard" is where the actual work happens in a SaaS. It's usually a complex mix of navigation, search, and data visualization.

## Learning Points
- **Sidebar + Main + Header**: Structuring a full-page application layout.
- **Complex Flex/Grid Combination**: Using Flexbox for the high-level layout and CSS Grid for the content cards.
- **Sticky Sidebar**: Ensuring the sidebar stays fixed while the content scrolls.
- **Responsive Dashboard**: How to handle sidebars and multi-column data on smaller screens.

## Why this matters for SaaS
Most B2B SaaS products are dashboards. Knowing how to build a robust, responsive skeleton is the most practical skill you'll learn in this course.

## How to run
1. Open `index.html`.
2. Notice how the sidebar takes the full height of the screen.
3. Try resizing the window to see how the "Stats Grid" and "Data Section" adapt.
