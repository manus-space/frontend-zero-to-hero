# Task 10: Feature Cards + Grid System

## Introduction
This task teaches you how to build a responsive feature cards grid using Tailwind CSS. You'll learn about Tailwind's grid system, card patterns, and hover effects that make UIs interactive and engaging.

## Learning Points
- **Tailwind Grid System**: Using `grid`, `grid-cols-1`, `md:grid-cols-2`, `lg:grid-cols-3` for responsive layouts.
- **Card Variants**: Creating consistent card components with borders, shadows, and rounded corners.
- **Hover Transforms**: Using `hover:shadow-2xl`, `hover:-translate-y-2` for interactive effects.
- **Group Hover**: Using `.group` and `group-hover:` to create effects that trigger on parent hover.
- **Color Gradients**: Building beautiful gradient headers for each card.

## Why this matters for SaaS
Feature cards are the backbone of SaaS landing pages. Learning to build them efficiently with Tailwind is essential for rapid development.

## How to run
```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Key Features
- 6 responsive feature cards with gradient headers
- Group hover effects for interactive cards
- Dark mode support
- Smooth transitions and transforms
- Mobile-first responsive design
