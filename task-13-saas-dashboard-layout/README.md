# Task 13: SaaS Dashboard Layout

## Introduction
This is the most complex layout you'll build. A real SaaS dashboard with sidebar, top navigation, stats cards, charts, and data tables. This teaches you how to structure a complete admin interface.

## Learning Points
- **Real SaaS Layout**: Sidebar + top nav + main content area.
- **Stats Cards**: Displaying key metrics with trends.
- **Data Tables**: Building responsive tables with status badges.
- **Complex Grid/Flex Combinations**: Combining CSS Grid and Flexbox for sophisticated layouts.
- **Tailwind Best Practices**: Using consistent spacing, colors, and components.

## Why this matters for SaaS
Most SaaS products are dashboards. Learning to build a professional dashboard layout is the most practical skill you'll learn. This is what you'll be building day-to-day as a frontend engineer.

## How to run
```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Key Features
- Responsive sidebar navigation
- Top navigation with search
- Stats cards with trend indicators
- Revenue chart placeholder
- Top plans breakdown
- Transaction table with status badges
- Dark mode support
- Fully responsive design
