# Task 09: Tailwind Hero + Responsive Navbar

## Introduction
Now we transition from pure CSS to **Tailwind CSS**, the modern utility-first framework that makes styling faster and more consistent. This task teaches you how to build a professional hero section and responsive navbar using Tailwind classes.

## Learning Points
- **Tailwind Utility Classes**: Using classes like `bg-blue-600`, `text-white`, `px-4` instead of writing CSS.
- **Responsive Design with Tailwind**: Using breakpoints like `md:`, `lg:` to make layouts responsive.
- **Dark Mode Support**: Leveraging Tailwind's `dark:` prefix for dark theme variants.
- **Group Hover Effects**: Using `group-hover` for interactive UI elements.
- **Gradient & Animations**: Modern visual effects using Tailwind's gradient and animation utilities.

## Why this matters for SaaS
Every modern SaaS uses Tailwind CSS. It's faster to develop, easier to maintain, and produces consistent, professional-looking UIs. Learning Tailwind is essential for modern frontend development.

## How to run
```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.

## Key Features
- Sticky responsive navbar with mobile menu button
- Hero section with gradient background
- Animated decorative elements
- Dark mode support (toggle dark mode in your browser)
- Responsive stats section
- Beautiful CTA buttons with hover effects
